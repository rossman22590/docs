## How to Use PolyCraft

PolyCraft offers two main input methods: Text to 3D and Image to 3D.

<Steps>
  <Step title="Choose Input Method" icon="upload">
    Decide whether you'll use a text prompt or upload an image as your starting point.
  </Step>
  <Step title="Provide Input" icon="pen-to-square">
    - For **Text to 3D**: Enter a text prompt.
    - For **Image to 3D**: Upload an image file.
  </Step>
  <Step title="Adjust Settings" icon="sliders">
    Configure parameters like **Model Version**, **Texture Quality**, **Style**, **Negative Prompt**, **Seed**, **Face Limit**, **Texture**, and **PBR**.
  </Step>
  <Step title="Generate" icon="play">
    Click the **Generate** button.
  </Step>
</Steps>

### Text to 3D Mode

Generate a 3D model from a text description.

<Steps>
  <Step title="Select Text to 3D" icon="keyboard">
    Ensure you are in Text to 3D mode (usually the default or selectable via a toggle/tab).
  </Step>
  <Step title="Enter Prompts" icon="pencil">
    Enter a **Prompt** describing the model. You can also add a **Negative Prompt** to exclude elements.
  </Step>
  <Step title="Adjust Settings" icon="sliders">
    Configure **Model Version**, **Texture Quality**, **Style**, **Model Seed (optional)**, **Face Limit**, **Texture**, and **PBR**.
  </Step>
  <Step title="Generate" icon="play">
    Click **Generate**.
  </Step>
</Steps>

### Image to 3D Mode

Generate a 3D model from a single image.

<Steps>
  <Step title="Select Image to 3D" icon="image">
    Choose Image to 3D mode (usually selectable via a toggle/tab).
  </Step>
  <Step title="Upload Image" icon="upload">
    Click **Choose File** and upload your image file (JPG or PNG).
  </Step>
  <Step title="Adjust Settings" icon="sliders">
    Configure **Model Version**, **Texture Quality**, **Style**, **Model Seed (optional)**, **Face Limit**, **Texture**, and **PBR**.
  </Step>
  <Step title="Generate" icon="play">
    Click **Generate**.
  </Step>
</Steps>

## Input Parameters and Options

PolyCraft offers various parameters for generation:

<ResponseField name="Prompt" type="string">
  (Text to 3D) Text description of the desired 3D model.
</ResponseField>
<ResponseField name="Or upload an image" type="file">
  (Image to 3D) Upload an image file (JPG or PNG).
</ResponseField>
<ResponseField name="Model Version" type="Enum">
  Choose the AI model version (e.g., `v2.5-20250123`, `v2.0-20240919`, `v1.4-20240625`).
</ResponseField>
<ResponseField name="Texture Quality" type="Enum">
  Choose the quality of the generated textures (`Standard`, `Detailed`).
</ResponseField>
<ResponseField name="Style" type="Enum">
  Select an artistic style (e.g., `Steampunk`, `Cartoon Character`, `Clay`).
</ResponseField>
<ResponseField name="Model Seed (optional)" type="string">
  Enter a seed for model consistency.
</ResponseField>
<ResponseField name="Negative Prompt" type="string">
  Text description for elements to avoid.
</ResponseField>
<ResponseField name="Face Limit" type="integer">
  Set the maximum number of faces (complexity).
</ResponseField>
<ResponseField name="Texture" type="boolean">
  Toggle to generate with textures.
</ResponseField>
<ResponseField name="PBR" type="boolean">
  Toggle to enable Physically Based Rendering textures.
</ResponseField>

## Post-Processing Options

PolyCraft offers several advanced post-processing options available from the history sidebar under each generated model.

### Animate Model

Add dynamic movement to a compatible generated model.

<Frame>
  <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/animate-format.png" alt="PolyCraft Animate Model Modal Screenshot" />
</Frame>

<Steps>
  <Step title="Select Animate" icon="play">
    From the history sidebar, click the **Animate** option under a generated model. This will open the Animate Model modal.
  </Step>
  <Step title="Adjust Settings" icon="sliders">
    Configure the animation parameters:
    - **Output Format**: Choose the output file format (GLB, FBX).
    <Frame>
      <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/animate-format.png" alt="PolyCraft Animate Output Format Screenshot" />
    </Frame>
    - **Animation Type**: Select a predefined animation (Idle, Walk, Climb, Jump, Run, Slash, Shoot, Hurt, Fall, Turn).
    <Frame>
      <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/animation-type.png" alt="PolyCraft Animation Type Screenshot" />
    </Frame>
  </Step>
  <Step title="Animate" icon="play">
    Click the **Animate** button in the modal. The AI will process the model to add the selected animation.
  </Step>
</Steps>

### Animate Parameters:

<ResponseField name="Output Format" type="Enum" required>
  The desired output file format for the animated model: `GLB`, `FBX`.
</ResponseField>
<ResponseField name="Animation Type" type="Enum" required>
  Select a predefined animation: `Idle`, `Walk`, `Climb`, `Jump`, `Run`, `Slash`, `Shoot`, `Hurt`, `Fall`, `Turn`.
</ResponseField>

### Stylize Model

Apply unique artistic styles to a generated model.

<Frame>
  <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/stylization-type.png" alt="PolyCraft Stylize Model Modal Screenshot" />
</Frame>

<Steps>
  <Step title="Select Stylize" icon="paintbrush">
    From the history sidebar, click the **Stylize** option under a generated model. This will open the Stylize Model modal.
  </Step>
  <Step title="Select Stylization Type" icon="list-ul">
    Choose the desired style from the **Stylization Type** dropdown: **Lego**, **Voxel**, **Voronoi**, or **Minecraft**.
  </Step>
  <Step title="Adjust Settings (Minecraft)" icon="sliders">
    If you select **Minecraft**, an additional setting for **Block Size** will be available.
  </Step>
  <Step title="Stylize" icon="play">
    Click the **Stylize** button in the modal. The AI will process the model to apply the selected style.
  </Step>
</Steps>

### Stylize Parameters:

<ResponseField name="Stylization Type" type="Enum" required>
  Select the artistic style: `Lego`, `Voxel`, `Voronoi`, `Minecraft`.
</ResponseField>
<ResponseField name="Block Size" type="integer">
  (If Stylization Type is `Minecraft`) Set the size of the blocks.
</ResponseField>

### Convert Model

Convert a model to various formats with options for remeshing, flattening, and scaling.

<Frame>
  <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/convert-modal.png" alt="PolyCraft Convert Model Modal Screenshot" />
</Frame>

<Steps>
  <Step title="Select Convert" icon="arrows-pointing-out">
    From the history sidebar, click the **Convert** option under a generated model. This will open the Convert Model modal.
  </Step>
  <Step title="Adjust Settings" icon="sliders">
    Configure the conversion parameters:
    - **Output Format**: Choose the output file format (GLTF, USDZ, FBX, OBJ, STL).
    <Frame>
      <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/convert-format.png" alt="PolyCraft Convert Output Format Screenshot" />
    </Frame>
    - **Enable Quad Remeshing**: Toggle to remesh into quads.
    - **Flatten Bottom**: Toggle to flatten the bottom of the model.
    - **Texture Size**: Choose the texture resolution (1024, 2048, 4096).
    <Frame>
      <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/convert-texture-size.png" alt="PolyCraft Convert Texture Size Screenshot" />
    </Frame>
    - **Texture Format**: Choose the texture image format (BMP, DPX, HDR, JPEG, OPEN_EXR, PNG, TARGA, TIFF, WEBP).
    <Frame>
      <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/convert-texture-format.png" alt="PolyCraft Convert Texture Format Screenshot" />
    </Frame>
    - **Set Pivot to Center Bottom**: Toggle to set the pivot point.
    - **Scale Factor**: Adjust the overall scale.
  </Step>
  <Step title="Convert" icon="play">
    Click the **Convert** button in the modal. The AI will process the model with the new settings.
  </Step>
</Steps>

<ResponseField name="Output Format" type="Enum" required>

</ResponseField>
<ResponseField name="Enable Quad Remeshing" type="boolean">
  Toggle to enable remeshing into quads.
</ResponseField>
<ResponseField name="Flatten Bottom" type="boolean">
  Toggle to flatten the bottom of the model.
</ResponseField>
<ResponseField name="Texture Size" type="Enum">
  Choose the texture resolution.
</ResponseField>
<ResponseField name="Texture Format" type="Enum">
  Choose the texture image format.
</ResponseField>
<ResponseField name="Set Pivot to Center Bottom" type="boolean">
  Toggle to set the pivot point to the center bottom.

  Adjust the overall scale of the model.

## Troubleshooting

If you encounter issues with PolyCraft, try these detailed solutions:

1. **Poor Model Quality**:
   - Refine your text prompts or use higher-quality input images. Adjust generation settings like Face Limit.
   - For Text to 3D, experiment with Model Version, Texture Quality, Style, Negative Prompt, and Seed.
   - For Image to 3D, ensure the subject is clear and well-lit.

2. **Generation Errors**:
   - Check your inputs for any unusual characters or overly complex descriptions.
   - Ensure image files are valid and within limits.

3. **Post-Processing Issues (Animate, Stylize, Convert)**:
   - Ensure the original model was generated successfully.
   - Check the parameters in the specific modal for valid inputs.
   - Some post-processing options may have specific requirements (e.g., Animation may work best on humanoid models).

4. **Download Issues**:
   - Ensure you have a stable internet connection.
   - Try a different format if one isn't working.
   - Clear your browser cache and try again.
