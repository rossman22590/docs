#### How to Use AI Tutor API

Follow these simple steps to integrate AI tutoring capabilities into your application using the AI Tutor API.

<AccordionGroup>
  <Accordion title="Step 1: Prepare Your Request" icon="pencil">
    Set up your API request with the necessary **headers and body content**. This includes your API key and the story or prompt you want to process.
  </Accordion>

  <Accordion title="Step 2: Send the API Request" icon="paper-plane">
    Use your preferred method (e.g., fetch, axios) to **send a POST request** to the AI Tutor API endpoint.
  </Accordion>

  <Accordion title="Step 3: Process the Response" icon="cogs">
    Once the API processes your input, it will return a **JSON response**. Handle this response in your application to extract the AI-generated tutoring content.
  </Accordion>

  <Accordion title="Step 4: Implement in Your Application" icon="laptop-code">
    Integrate the AI-generated content into your application's **user interface or learning flow**.
  </Accordion>
</AccordionGroup>
