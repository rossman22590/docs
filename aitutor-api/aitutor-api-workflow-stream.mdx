# Streaming API Documentation

## Overview

Our AI Tutor API supports real-time data streaming to enhance user experience and provide faster data retrieval. This guide explains how to implement client-side streaming requests using our API.

<Note>
  This API is designed to work seamlessly with modern frontend frameworks and provides real-time streaming capabilities.
</Note>

### Getting Started

First, generate a token:

```typescript
const response = await fetch('https://aitutor-api.vercel.app/api/v1/token/generate');
const { token } = await response.json();
```

## Making Requests

<CodeGroup>

## React Implementation

```tsx StreamingText.tsx
"use client";

import { useCallback, useEffect, useState } from "react";
import { Spinner } from "./loaders";

interface StreamingTextProps {
  url: string;
  fallbackText: string;
  className?: string;
}

export default function StreamingText({
  url,
  fallbackText,
  className,
}: StreamingTextProps) {
  const [loading, setLoading] = useState(false);
  const [result, setResult] = useState("");

  const getData = useCallback(async () => {
    setLoading(true);
    try {
      const response = await fetch(url, {
        method: "POST",
        headers: {
          "Content-Type": "application/json",
        },
      });
      if (!response.ok) {
        throw new Error(`HTTP error! status: ${response.status}`);
      }

      const data = response.body;
      if (!data) {
        setResult(fallbackText);
        return;
      }

      const reader = data.getReader();
      const decoder = new TextDecoder();
      let done = false;
      setLoading(false);

      while (!done) {
        const { value, done: doneReading } = await reader.read();
        done = doneReading;
        const chunkValue = decoder.decode(value);
        setResult((prev) => (prev ?? "") + chunkValue);
      }
    } catch (error) {
      console.error('Streaming error:', error);
      setResult(fallbackText);
    } finally {
      setLoading(false);
    }
  }, [url, fallbackText]);

  useEffect(() => {
    if (url) {
      getData();
    }
  }, [getData, url]);

  return loading ? (
    <Spinner className={className} />
  ) : (
    <p className={className}>{result}</p>
  );
}
```

```tsx
import StreamingText from './StreamingText';

function MyComponent() {
  return (
    <StreamingText
      url="https://aitutor-api.vercel.app/api/v1/chat/your-token/stream"
      fallbackText="Loading failed. Please try again."
      className="text-lg text-gray-800"
    />
  );
}
```
