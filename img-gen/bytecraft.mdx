## Overview

ByteCraft is a cutting-edge AI platform that leverages ByteDance's powerful **Seedream 4.0 models** for both image generation and advanced editing. With its ability to produce high-resolution images and handle a wide range of styles, ByteCraft is the perfect tool for creating posters, infographics, and other professional-quality visuals.

## Modes of Operation

ByteCraft offers two primary modes of operation: **Generate** and **Edit**.

### Generate Mode

In Generate mode, you can create new images from scratch using text prompts.

<AccordionGroup>
  <Accordion title="Parameters" icon="sliders">
    - **Prompt**: Your text description of the desired image.
    - **Aspect Ratio**: Choose from a variety of preset aspect ratios.
    - **Number of Images**: Generate up to four images at once.
    - **Seed**: Use a seed to control the output. The same seed + same prompt will produce an identical output.
  </Accordion>
</AccordionGroup>

### Edit Mode

In Edit mode, you can modify existing images using text prompts.

<AccordionGroup>
  <Accordion title="Parameters" icon="sliders">
    - **Prompt**: Your text description of the desired edits.
    - **Seed**: Use a seed to control the output. The same seed + same prompt will produce an identical output.
  </Accordion>
</AccordionGroup>

To achieve the best results with ByteCraft, focus on descriptive and specific prompts.

### Prompting Techniques:

<AccordionGroup>
  <Accordion title="Be Clear and Specific" icon="pencil">
