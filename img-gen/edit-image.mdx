## How to Use Edit Image

Follow these steps to modify your images using maskless AI editing:

<Steps>
  <Step title="Upload Your Image" icon="upload">
    Upload the base image you wish to edit. This is the image that the AI will modify based on your instructions.
  </Step>

  <Step title="Describe the Changes" icon="pencil">
    In the main prompt field, provide a clear text description of the **modifications you want to make** to the image.
    
    <Callout type="example">
      *Examples:*
      - "change the outfit to a blue jacket"
      - "add a mustache to the person's face"
      - "turn the sky into a starry night"
    </Callout>
  </Step>

  <Step title="Add Negative Prompt (Optional)" icon="ban">
    Use the negative prompt field to specify any elements or characteristics you want to **avoid** in the edited image.
    
    <Callout type="info">
      This helps prevent unwanted changes or artifacts.
    </Callout>
  </Step>

  <Step title="Adjust Settings" icon="sliders">
    Configure the generation parameters to fine-tune the editing process:
    
    <AccordionGroup>
      <Accordion title="Guidance Scale" icon="compass">
        Controls how closely the AI adheres to your description of changes. Higher values result in changes that more strictly follow your prompt.
      </Accordion>
      <Accordion title="Inference Steps" icon="shoe-prints">
        Determines the number of denoising steps. Higher values can lead to more detailed and refined edits.
      </Accordion>
      <Accordion title="Seed" icon="seedling">
        Enter a specific number for reproducible results. Useful for applying similar edits consistently or iterating on a specific modification.
      </Accordion>
    </AccordionGroup>
  </Step>

  <Step title="Generate Edited Image" icon="play">
    Click the "Generate" button to apply the described changes to your image using the Step1X-Edit model.
  </Step>
</Steps>

## Input Parameters and Options

Here are the key input parameters and options available in Edit Image:

<ResponseField name="image_upload" type="file" required>
  The base image file to be edited.
</ResponseField>

<ResponseField name="description_of_changes" type="string" required>
  A text prompt describing the desired modifications to the image.
</ResponseField>

<ResponseField name="negative_prompt" type="string">
  A text prompt specifying elements or characteristics to avoid in the edited image. (Optional)
</ResponseField>

<ResponseField name="guidance_scale" type="float">
  Controls how closely the AI adheres to the description of changes.
</ResponseField>

<ResponseField name="inference_steps" type="integer">
  The number of inference steps to perform.
</ResponseField>

<ResponseField name="seed" type="integer">
  A seed value for reproducible results. (Optional)
</ResponseField>
