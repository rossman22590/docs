#### How to Use FluxCraft

Follow these steps to harness the full power of FluxCraft for your image generation projects.

<AccordionGroup>
  <Accordion title="Step 1: Select Your Model" icon="list">
    Choose your preferred model from the dropdown menu:
    - Speed: Flux Schnell (fastest generation)
    - Balanced: Flux Dev (includes image-to-image option)
    - Pro 1.1: Flux Pro 1.1 (high-quality outputs)
    - Realism: Flux Realism LoRA (for realistic images)
    - Custom: Flux Dev with Custom LoRAs
    - Multi: Flux Dev with Multi LoRA support

    ![Model Selection](https://img.mytsi.org/i/yoZh738.png)
  </Accordion>
  <Accordion title="Step 2: Input Your Prompt" icon="keyboard">
    Enter a **detailed prompt** in the text area to guide the AI in generating your desired image.
  </Accordion>
  <Accordion title="Step 3: Adjust Generation Settings" icon="sliders">
    Click the **Settings** button to access and modify various parameters:
    - Image Size: Choose from preset sizes like square, portrait, or landscape
    - Inference Steps: Control the detail level (more steps = more detail but slower generation)
    - Seed: Set a specific seed for reproducible results or use -1 for random
    - Guidance Scale: Adjust how closely the AI follows your prompt
    - Strength: (for Multi model) Adjust the influence of the base model

    ![Generation Settings](https://img.mytsi.org/i/o3Th739.png)
  </Accordion>
  <Accordion title="Step 4: Configure Advanced Options" icon="gears">
    For Custom and Multi models, add and configure LoRAs:
    - Click "Add LoRA" to include custom LoRAs
    - Set LoRA URL, trigger word (for Multi), and scale for each LoRA
    
    For Balanced model, use Image-to-Image feature:
    - Toggle "Image to Image" switch
    - Upload a base image and set image strength

    ![Image-to-Image Option](https://img.mytsi.org/i/K9pT740.png)
  </Accordion>
  <Accordion title="Step 5: Generate and View" icon="play">
    Click **Generate** and wait for the process to complete. The generated image will appear in the gallery below.

    ![Generation in Progress](https://img.mytsi.org/i/9Nnp741.png)
    ![Generated Image Result](https://img.mytsi.org/i/J5hk742.png)
  </Accordion>
</AccordionGroup>

#### Managing Your Generated Images

<AccordionGroup>
  <Accordion title="Viewing and Downloading" icon="eye">
    - Click on an image in the gallery to view it in full size
    - Use the "Download" button to save individual images
    - Click "Download All Images" to save all generated images as a zip file
  </Accordion>
  <Accordion title="Image Details" icon="circle-info">
    When viewing a full-size image, you can see details such as:
    - The prompt used
    - Image size
    - Inference steps
    - Seed
    - Guidance scale
  </Accordion>
  <Accordion title="Additional Actions" icon="ellipsis">
    - Copy the prompt of any image to reuse or modify
    - Send images directly to a gallery (via Discord integration)
    - Clear all generated images from the gallery
  </Accordion>
</AccordionGroup>
