# Error Reference Guide

This document provides a list of common errors that users might encounter when using Machine, along with their causes and solutions.

## Common Error Codes

<ResponseField name="404" type="Not Found">
  **Resource Not Found**
  
  The requested resource could not be found or has been removed.
  
  **Solution**: 
  - Check that you're using the correct URL
  - Try refreshing the page
  - Return to the dashboard and try accessing the resource again
</ResponseField>

<ResponseField name="422" type="Unprocessable Entity">
  **Request Error**
  
  Machine couldn't process your request due to validation issues.
  
  **Solution**: 
  - Try simplifying your request
  - Check that any uploaded files meet our requirements
  - If asking a complex question, break it into smaller parts
</ResponseField>

## Account & Subscription Errors

### "Invalid subscription tier" Error

**Cause**: Your subscription appears to be invalid or has encountered a processing issue.

**Solution**:
1. Go to Account Settings â†’ Subscription to verify your current plan
2. Try refreshing your browser or logging out and back in
3. Contact customer support if the issue persists

### Checkout Problems

**Cause**: Issues during the payment or subscription upgrade process.

**Solution**:
1. Try a different browser or clear your browser cache
2. Ensure your payment method is valid and has sufficient funds
3. Use our direct checkout link from your account page

## Usage Problems

### "Session Expired" Error

**Cause**: Your session has timed out due to inactivity or system maintenance.

**Solution**:
1. Refresh the page
2. Log out and log back in
3. If the problem continues, wait a few minutes and try again

### "File Upload Failed" Error

**Cause**: Issues with file uploads can occur for several reasons.

**Solution**:
1. Ensure your file is under the 25MB size limit
2. Check that you're using a supported file format
3. Try uploading again or use a different file

### "Chat Not Responding" Error

**Cause**: The AI assistant might be processing a complex request or experiencing high demand.

**Solution**:
1. Wait a few moments for the response
2. If no response after 30 seconds, refresh the page
3. Start a new chat if the issue persists

## Feature Access Issues

### "Feature Not Available" Error

**Cause**: You've attempted to use a feature not included in your current subscription plan.

**Solution**:
1. Check your current subscription level in Account Settings
2. Upgrade your plan to access premium features
3. See our pricing page for feature comparison

### "Usage Limit Reached" Error

**Cause**: You've reached the maximum usage for your current subscription period.

**Solution**:
1. Check your usage statistics in Account Settings
2. Wait until your usage resets on your billing date
3. Consider upgrading to a plan with higher limits
