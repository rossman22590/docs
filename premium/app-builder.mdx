## 1. Overview

App Builder Beta is a powerful tool designed to streamline the process of creating applications. With an intuitive interface and advanced features, it allows developers of all skill levels to build, preview, and deploy applications with ease.

## 2. Getting Started

To begin using App Builder Beta, navigate to the Premium dashboard in AI Tutor.

## 3. Choosing a Persona

One of the unique features of App Builder Beta is the ability to choose a persona for your project.

![Choosing a Persona](https://img.mytsi.org/i/qlhe733.png)

- Click on the "Choose Persona" dropdown menu
- Select from a variety of pre-defined personas, each tailored to specific project types
- Your chosen persona will guide the AI in generating more relevant and context-appropriate code

## 4. Selecting a Language Model

App Builder Beta integrates with various Language Models (LLMs) to enhance your development experience.

![Selecting a LLM](https://img.mytsi.org/i/FJGv728.png)

- Navigate to the "Select LLM" section
- Choose from the available LLMs based on your project requirements
- The selected LLM will be used for code generation and chat interactions

## 5. Configuring Parameters

Customize your project by configuring various parameters to suit your needs.

![Configuring Parameters](https://img.mytsi.org/i/0beo727.png)

- Access the "Parameters" section
- Adjust settings such as project type, target platform, and specific features
- These parameters will guide the AI in generating appropriate code and structure for your project

## 6. Viewing Code and Preview

App Builder Beta offers two main views for your project: Code and Preview.

### 6.1 Code View

![Code View](https://img.mytsi.org/i/lWKT736.png)

- Access the generated code for your project
- Options to copy code snippets or download entire files
- Syntax highlighting for easy readability

### 6.2 Preview

![Preview](https://img.mytsi.org/i/wgD2735.png)

- Quickly view a live preview of your project
- Interact with your application in real-time
- Test functionality and design without leaving the platform

## 7. Deploying to the Cloud

App Builder Beta simplifies the deployment process with its built-in cloud deployment feature.

### 7.1 Initiating Deployment

![Deploy to Cloud Button](https://img.mytsi.org/i/q9gS731.png)

- Click the "Deploy to Cloud" button to start the deployment process

### 7.2 Setting Expiration Date

![Set Expiration Date](https://img.mytsi.org/i/qYg5732.png)

- Choose an expiration date for your deployed application
- This feature helps manage resources and ensures your deployments are up-to-date

### 7.3 Finalizing Deployment

- Review the deployment settings
- Click "Accept and Deploy"
- Once deployment is complete, you'll receive a URL to access your live application
