## 14. FAQ

[Previous content remains, with these additions:]

**Q: How does the new file upload feature work?**
A: The file upload feature supports multiple file formats. Simply click the upload button, select your file, and the AI will analyze and incorporate the content into your conversation.

**Q: Can I switch between branched conversations?**
A: Yes, you can easily switch between branched conversations using the branch navigation feature.

**Q: How does the split-screen feature enhance my workflow?**
A: The split-screen feature allows you to engage in multiple conversations or tasks simultaneously, improving productivity and enabling easy comparison of different AI outputs.

**Q: What types of diagrams can be visualized in Beam 2 Beta?**
A: Beam 2 Beta supports various diagram types, including flowcharts, mind maps, and technical diagrams. The AI can both interpret and generate these visual representations.
