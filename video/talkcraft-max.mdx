# TalkCraft Max: AI-Powered Talking Head Videos

<Tip>
  Bring your characters to life with TalkCraft Max! Upload an image and an audio file to create realistic, audio-driven animations.
</Tip>

## Overview

TalkCraft Max, powered by Hedra Character 3, is a state-of-the-art tool for creating realistic talking head videos. It takes an image of a character and uses audio-driven animation to make it speak. You can also provide an optional prompt to guide the generation and improve expressiveness.

## Features

<CardGroup cols={2}>
  <Card title="Audio-Driven Animation" icon="waveform">
    Animate a character's face using an audio file up to 5 minutes long.
  </Card>
  <Card title="Expressive Control" icon="face-smile">
    Use an optional prompt to guide the character's expressions and emotions.
  </Card>
  <Card title="High-Quality Output" icon="star">
    Generate high-quality videos with realistic lip-syncing and facial movements.
  </Card>
  <Card title="Customizable Aspect Ratio" icon="crop">
    Choose between 1:1, 16:9, and 9:16 aspect ratios to fit your needs.
  </Card>
</CardGroup>

## How to Use TalkCraft Max

<Steps>
  <Step title="Select Audio">
    Upload an audio file up to 5 minutes long. Make sure the audio is clear and only has one speaker.
    <Frame>
      <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/talkmax1.png" alt="Select Audio" />
    </Frame>
  </Step>
  <Step title="Select Portrait">
    Upload a portrait of your character. Make sure the face is clearly visible, the mouth is closed, and the eyes are open. A neutral facial expression is preferred for the most accurate results.
    <Frame>
      <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/talkmax2.png" alt="Select Portrait" />
    </Frame>
  </Step>
  <Step title="Select Aspect Ratio and Add Optional Prompt">
    Choose the correct aspect ratio for your video and provide an optional prompt to guide the generation. Then click "Generate".
    <Frame>
      <img src="https://pixiomedia.nyc3.digitaloceanspaces.com/docs/talkmax3.png" alt="Select Aspect Ratio and Add Prompt" />
    </Frame>
  </Step>
</Steps>
